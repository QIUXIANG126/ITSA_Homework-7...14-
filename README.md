# [輾轉相除法](itsa_Euclidean_algorithm.cpp)
用遞迴函式。
回傳時將 b 的數值傳給a 再將 a%b 傳給 b 直到 a%b=0時回傳 a值，因為回傳時a 的數值是b ，b的數值是a%b。 當a%b=0表示 a是b的倍數 則a 就是a b 的最大公因數。
